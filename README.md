# مؤسسة المنى الإبداعية — Al-Muna Foundation

منصة رقمية مؤسسية وأرشيفية لمؤسسة المنى الإبداعية، تعمل كواجهة رسمية للهوية المؤسسية وكمستودع رقمي موثق للإنتاج الأدبي.

## المميزات

- **أرشيف رقمي** للنصوص الأدبية (شعر، قصة، مقال، رواية، نثر)
- **نظام بحث** دقيق مع فلترة متعددة (النوع، التصنيف، الكاتب)
- **ملفات شخصية** للمبدعين والكتّاب
- **نظام إدارة محتوى (CMS)** متكامل بلوحة تحكم
- **حماية الملكية الفكرية** (منع النسخ، علامة مائية)
- **تصميم عصري** بالوضع الليلي الداكن
- **متجاوب** مع جميع الأجهزة (RTL)

## التقنيات المستخدمة

| الطبقة | التقنية |
|--------|---------|
| **Frontend** | React 18 + TypeScript + Vite |
| **Styling** | Tailwind CSS |
| **Backend** | Express.js |
| **Database** | SQLite (sql.js) |
| **Auth** | JWT + bcrypt |
| **Hosting** | Railway |

## البدء السريع

```bash
# تثبيت الاعتماديات
pnpm install

# بناء الواجهة الأمامية
pnpm build

# تشغيل الخادم
pnpm start
```

## المسارات

| المسار | الصفحة |
|--------|--------|
| `/` | الرئيسية |
| `/archive` | الأرشيف الأدبي |
| `/texts/:slug` | تفاصيل النص |
| `/creators` | المبدعون |
| `/creators/:slug` | ملف المبدع |
| `/events` | الفعاليات |
| `/search` | البحث |
| `/about` | عن المؤسسة |
| `/contact` | اتصل بنا |
| `/admin` | لوحة التحكم |

## API

| المسار | الوصف |
|--------|-------|
| `GET /api/texts` | قائمة النصوص |
| `GET /api/texts/:slug` | تفاصيل النص |
| `GET /api/authors` | قائمة المؤلفين |
| `GET /api/search` | بحث متقدم |
| `GET /api/search/suggest` | اقتراحات البحث |
| `GET /api/categories` | التصنيفات |
| `GET /api/events` | الفعاليات |
| `POST /api/contact` | إرسال رسالة |
